# Ilana Teman – Artist Portfolio

Static portfolio site for [ilana-teman-art.co.il](https://ilana-teman-art.co.il).

## Running locally

The site is plain HTML/CSS/JS with no build step. You just need a local web server.

**Option A — Python (built-in on macOS/Linux):**

```bash
cd /path/to/ilana-teman-art
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

**Option B — Node (if you have it):**

```bash
npx serve .
```

**Option C — VS Code:**

Install the "Live Server" extension, right-click `index.html` → *Open with Live Server*.
